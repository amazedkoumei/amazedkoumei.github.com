有名人のホーム画面を眺められるTeetJackというサイトを公開しました。

Tweetjack あのひとのホーム画面をリアルタイムウォッチ

あの人のホーム画面をリアルタイムウォッチ


ホーム画面とはTwitterにログインした直後に表示される画面のことです。
フォローしているアカウントは人それぞれなので、ホーム画面というのは百人百様です。

他人のホーム画面に興味はありませんか？
私は興味津々です。

なのでまずは私が一番ホーム画面を覗いみたいと思った方
@takapon_jpこと堀江貴文さんのホーム画面をジャックしてみることにしました。

私は堀江貴文さんの発言をよく追いかけている方だと思いますが、Twitterの登場以降、従来RSSリーダで得ていた情報のほとんどはTwitterから得るようなったとよくおっしゃっています。

そんな堀江貴文さんのTwitterホーム画面に興味のある方は一緒に覗いてみましょう。

本題に入る前に一言 - 先日の判決について

先日(2011/04/26) 堀江貴文さんに下された実刑判決、あの時もうすでにTweetJackを作り始めていたこともあり、本当に驚きました。

ライブドア事件後、彼が積極的に情報発信されたこともあり、日本の司法についてはインターネット上でも、その他の媒体でも、多くの事が論じられています。

堀江さんの立場での見解、反対の立場からの見解、全くの異なる立場からの見解、そのどれもに目を通して自分の国の司法というものを、この機会に自分なりに考えてみようと思いました。

考えることが国民の義務だとは言いませんが、考えないことで損をするのはたぶん自分だなと。


と、そんなことを思いつつ本題です。

TweetJackの推奨環境

Chrome, Firefox, Safari の最新バージョンを推奨します。
iPhoneのSafariでも動作します。

IEは非推奨です。

むしろ爆発してください。

使い方

  1. TweetJackのTOPにアクセスし、「@takapon_jpのホーム画面を見る」ボタンをクリックします。
  2. @takapon_jpがフォローしている人のつぶやきが表示されます。
  3. しばらく待つと・・
  4. @takapon_jpがフォローしている人がつぶやくごとにタイムラインが更新されていきます。
  5. ユーザ名をクリックするとその人のアカウントに、日時をクリックするとそのつぶやきにジャンプします。
    フォローするも良し、リツイートするもまた良しです。私と一緒にお楽しみいただけると幸いです。

 

使用している外部サービス

TweetJackではTwitter Streaming API 、 Pusherという外部サービスを使用しています。

Twitter Streaming API はTwitterに投稿されるつぶやきをリアルタイムに取得できるTwitter社のサービス、PusherはWebSocketを代替してくれるサービスです。

Pusherは将来的にはサービスに制限を設けることでのマネタイズを考えているようですが
現在β公開中ということで、無料で無制限版を公開してくれています。

使ってみて、またドキュメントを読んでみて大変気持ちのいいサービスでしたので、いつか紹介エントリを書きたいと思わせてくれるサービスでした。


一方でこれら外部サービスを使っていることが不安の種だったりもします。
思わぬ制限事項があったりしますので・・。

もしタイムラインが更新されない等の事象がありましたら@amazedkoumeiまでご連絡いただけると助かります。

最後に

TweetJackの情報はどんどん流れていきます。
すきま時間を埋めるには適していますが、漏れなくつぶやきを捕捉するという使い方には不向きです。

もし、他人のホーム画面を漏れなく捕捉したいのだ、という方がいらっしゃいましたら
Ironsさんが公開しているTwitter Timeline ViewerというYahoo Pipesがオススメです。

Twitter Timeline Viewerは他人のホーム画面をRSSフィードに変換してくれるのでRSSリーダで捕捉することができて、私自身も重宝させいていただいてます。

興味のある方は、ぜひ上のリンクからアクセスしてみてください。
